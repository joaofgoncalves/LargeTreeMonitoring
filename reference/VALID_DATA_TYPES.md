# Supported time-series columns for break detection

Character vector containing column names that break-detection wrappers
accept through their `ts_name` argument. These identify the raw or
preprocessed spectral-index series stored in an `spidf` object.

## Usage

``` r
VALID_DATA_TYPES
```

## Format

A character vector.
