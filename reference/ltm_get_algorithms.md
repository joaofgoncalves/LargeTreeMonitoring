# List algorithms stored in a `ts_breaks` container

List algorithms stored in a `ts_breaks` container

## Usage

``` r
ltm_get_algorithms(ts_breaks_obj)
```

## Arguments

- ts_breaks_obj:

  An object of class `ts_breaks`.

## Value

Character vector of algorithm names.
